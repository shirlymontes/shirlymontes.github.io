# Personal Portfolio Website  
**Developer:** Shirly Rose Montes  
**Course:** BS in Information Technology – Software Development Section S3101 
**Project Type:** Individual – Web Development  

---

## Project Overview
This project is a **personal portfolio website** developed as part of my coursework.  
Its primary goal is to build a responsive, accessible, and aesthetically pleasing online portfolio that demonstrates my technical and creative abilities in front-end development and design.  

The site serves as a foundational step toward establishing my professional identity as a future front-end developer and UI/UX designer.  

---

## Objectives
1. **Professional Representation** – Create a clean, user-friendly personal website that highlights my skills and background.  
2. **Responsive Design** – Ensure compatibility and visual consistency across different devices (desktop, tablet, and mobile).  
3. **Functional Navigation** – Implement smooth scrolling, sticky navbar, and responsive layout.  
4. **Foundational JavaScript Integration** – Use JavaScript to enhance interactivity (scroll effects, form validation, etc.).  
5. **Preparation for Real Portfolio** – Structure the layout to be expandable when actual project samples and case studies are ready.  

---

## Scope
The website includes the following key sections:

**Home (Hero)** - Introduces the developer with a tagline and a call-to-action button. 
**Projects** - Currently includes placeholder cards; planned to feature real projects in future iterations. 
**Skills** - Displays core technical and design skills using responsive icons and layout. 
| **Contact** - Contains a simple form and social media links for inquiries and networking. 

---

## Features Implemented
- Responsive layout using **Flexbox** and **CSS media queries**.  
- **Sticky Navbar** that remains visible during scrolling.  
- **Smooth Scrolling** behavior for internal navigation links.  
- Basic **JavaScript enhancements** (scroll effects, button interactions).  
- Structured and organized folder hierarchy following professional web standards.  

---

## Technology Stack
- **HTML5** — Semantic and structured markup.  
- **CSS3** — Styling, Flexbox, responsive design, and hover effects.  
- **JavaScript (ES6)** — Interactive components and UI logic.  
- **GitHub Pages** — For hosting and deployment.
